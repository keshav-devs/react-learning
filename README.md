# React Learning

This repository documents my journey of learning **React.js**, a popular JavaScript library used to build modern and interactive user interfaces.

It contains practice examples, concept demonstrations, and small projects built while exploring the fundamentals of React and frontend development.

---

## Purpose

- Learn and understand core React concepts  
- Practice building reusable UI components  
- Strengthen frontend development skills  
- Build small projects while learning React  

---

## Topics Covered

The repository includes examples and practice related to important React concepts such as:

- JSX
- React Components
- Props
- State
- Event Handling
- Conditional Rendering
- Lists and Keys
- Forms in React
- React Hooks
- useState
- useEffect
- Component Reusability
- Basic React Project Structure

More topics will be added as the learning journey progresses.
